# NextjsWP

This project is a Next.js web application using Tailwind CSS for styling. It features a modular structure with multiple pages and components, including a custom navigation bar and several content sections.

## Project Structure

- `src/app/` — Main application pages and subfolders for different sections (e.g., township, homes, event system, showcase, contact, etc.)
- `src/components/` — Reusable React components (e.g., Navbar)
- `src/lib/` — Utility libraries (e.g., Apollo Client setup)
- `public/` — Static assets (images, etc.)
- `globals.css` — Global styles

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Run the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view the app.

## Features
- Responsive navigation menu with animated hamburger icon
- Multiple content pages (Homepage, Township, Homes, Event System)
- Tailwind CSS for rapid UI development
- Modular and scalable folder structure

## Customization
- Add or modify pages in `src/app/`
- Update navigation links in `src/components/Navbar.jsx`
- Add new components in `src/components/`
